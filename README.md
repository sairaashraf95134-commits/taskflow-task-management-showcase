# TaskFlow - Task Management Web App

TaskFlow is a modern task management web application developed using Replit.

## Live Demo
https://task-flow-dashboard--sairaashraf9513.replit.app/

## Features
- Add new tasks
- Display all tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by status
- Progress overview dashboard
- Responsive design for desktop/mobile

## Technologies Used
- HTML
- CSS
- JavaScript
- Replit

## Project Objective
This project was created as part of my Agentic AI / AI Powered Software Engineer course to demonstrate frontend development and web app creation using Replit.
